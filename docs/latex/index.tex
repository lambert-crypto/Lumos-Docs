A 2-\/D game engine written in C++.\hypertarget{index_autotoc_md1}{}\doxysection{How to run}\label{index_autotoc_md1}

\begin{DoxyEnumerate}
\item Run {\ttfamily meson setup build} to create the build directory
\item Go to build directory with {\ttfamily cd build}
\item Run {\ttfamily meson compile \&\& ./debug}
\item To change target file change the same in {\ttfamily meson.\+build}
\end{DoxyEnumerate}\hypertarget{index_autotoc_md2}{}\doxysection{Dependencies}\label{index_autotoc_md2}

\begin{DoxyEnumerate}
\item For GLFW and GLEW\+: {\ttfamily sudo apt-\/get install libglfw3-\/dev libglew-\/dev}
\item For spdlog\+: {\ttfamily sudo apt-\/get install libspdlog-\/dev}
\item glm version {\ttfamily 0.\+9.\+9.\+8} (pre installed as header library in {\ttfamily Lumos} directory)
\end{DoxyEnumerate}\hypertarget{index_autotoc_md3}{}\doxysection{File structure}\label{index_autotoc_md3}

\begin{DoxyEnumerate}
\item The engine code is present in {\ttfamily Lumos} directory
\item Testing out feature can be done by making another file in {\ttfamily Test} folder (change the target in {\ttfamily meson.\+build})
